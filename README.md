# perl-ebpf

ebpf - It's pure-perl ebpf loader

# SYNOPSIS

    use ebpf;

# DESCRIPTION

The goal of this project is to enable the attachment of eBPF programs from Perl, create eBPF maps, and perform operations on those eBPF maps.
